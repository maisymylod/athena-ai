# Example images for the demo

These six images power the "Try with these" affordance on the landing page. The
six files currently checked in are programmatically-generated placeholder
gradients with PLACEHOLDER labels — they exist only so the UI renders during
development.

**Before launch, replace each placeholder file with a real example:**

- `real_1.jpg`, `real_2.jpg`, `real_3.jpg` — three CC0-licensed photos
  (Unsplash is the easiest source). Aim for a portrait, a landscape, and a
  group/scene to show the model isn't only good at one type of image.
- `ai_1.jpg`, `ai_2.jpg`, `ai_3.jpg` — three publicly-shared AI-generated
  samples. Stable Diffusion or Midjourney outputs from a community gallery
  with permissive use are fine.

Then update `manifest.json` to reflect the new credits/labels. Filenames must
stay the same so the manifest doesn't need restructuring.
